## インストール

```bash
$ npm i docsify-cli -g
```


## ローカル起動


```bash
$ docsify serve ./docs/
```

## textlint


```bash
$ yarn textlint
```

## 編集履歴の更新


```bash
$ yarn changeLog
```

## 本番環境デプロイ

```bash
$ git checkout main
$ git pull --ff-only origin main
$ git tag -a v1.0.0 -m 'リリース内容'
$ git push origin v1.0.0
```
