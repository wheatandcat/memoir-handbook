# 画面名

記載時に使用する画面のブレを無くすため以下に画面を記載している。

## Top画面
 - ログイン前の初期画面

## ログイン画面
 - ログイン時の情報入力する

## 達成したタスクの登録画面
 - 達成したタスクを登録する
 - [達成したタスク](functions/task/01-task)

## 達成したタスクの表示画面
 - 達成したタスクを表示する

## 振り返り画面
 - 1週間の振り返りをする
 - [振り返り機能](functions/memoir/01-memoir)

## マイページ画面
 - プロフィールを編集する
 - 共有メンバーを表示する
